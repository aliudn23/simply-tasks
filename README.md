# Simply Task
Simply Task is a simple and efficient tool for managing your daily tasks. The application provides an intuitive interface to help you organize, track, and update your to-do items with ease.

**Design:**
The application is designed with simplicity and efficiency in mind. By focusing on an intuitive user interface and streamlined workflows, users can manage their tasks quickly and easily, minimizing the time to create this application.

**Authentication Process:**  
To get started, you log in using a static bearer token. This initial token is exchanged by the application for a secure session token, which is then used for all further interactions with the task management API.

Username : user@example.com
Password : password

**Task Management Features:**  
Once authenticated, you can create new tasks, view your existing tasks, update their details, and delete tasks you no longer need. All these operations are performed securely through the external API.
